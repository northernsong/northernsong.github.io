spring boot security入门级使用

之前用的最多的是shiro，简单好理解，文档多。spring security相对好像评论差一点，但既然决定使用spring全家桶，就都要体验体验，实际上，spring security用起来也挺简单的。

项目说明

spring security 功能远不止页面的访问控制，在spring boot的示例使用security 也是通过注解控制访问方法的。在这里只演示页面的访问控制。
项目中有3个页面，首页（/）、登录页(/login)任何人都可以访问，hello页模拟业务页，只能登录后访问。
